# Módulo 3: Collaboration Features

Finalizamos as aulas do Módulo 3 e agora é hora de colocar em prática!

A atividade a seguir visa consolidar o conhecimento apresentado na aula, abordando os seguintes tópicos:

- Criar repositório (Não é necessário criar outro repositório caso já tenha criado o repositório da atividade do módulo 2)
- Criar e Gerenciar Issues
- Criar Discussions
- Trabalhar com Pull Requests
- Criar e Gerenciar Wikis

## Criando Repositório

**Requisitos:**
- Conta GitHub

**Procedimento:**

Acesse [github.com](https://github.com) no seu navegador e faça login em sua conta (clique no botão "Sign In" localizado no canto superior direito)

Após fazer login, clique no avatar no canto superior direito, selecione "Your repositories" e depois clique em "New"

![Imagem 1](https://files.cdn.thinkific.com/file_uploads/401289/images/2b2/7a6/aa2/1716920476197.png)

Nomeie o repositório como `learn-github-SeuNome`, e clique em "**Create repository**".

![Imagem 3](https://files.cdn.thinkific.com/file_uploads/401289/images/0ef/bb9/917/1716920575125.png)

Após clicar em **create repository** deveremos ter esse resultado:

![Imagem 4](https://files.cdn.thinkific.com/file_uploads/401289/images/6e4/86f/008/1716920603018.png)

Agora vamos no clicar na opção **Branch** na tela inicial do repositório

![Imagem 5](https://files.cdn.thinkific.com/file_uploads/401289/images/948/6a9/472/1716930191259.png)

Em seguida vamos clicar na opção **New branch (lado direito superior)** para criar uma nova branch

![Imagem 6](https://files.cdn.thinkific.com/file_uploads/401289/images/5a6/c60/c4d/1716930236583.png)

Em seguida vamos dar um nome para nova branch, pode ser o seu proprio nome e clicar em **create new branch**

O Resultado deverá ser assim. Aparecendo a nova branch criada com o nome que você escolheu.

![Imagem 8](https://files.cdn.thinkific.com/file_uploads/401289/images/407/76e/0e9/1716930328017.png)

Agora clique em cima do nome da nova branch e vamos criar um arquivo dentro dela.

![Imagem 9](https://files.cdn.thinkific.com/file_uploads/401289/images/e79/d7a/903/1716930356435.png)

Clique na opção Add file e em seguida create new file

![Imagem 10](https://files.cdn.thinkific.com/file_uploads/401289/images/a99/f0c/0bb/1716930384697.png)

No campo Name your file escreva: arquivo.md e no campo abaixo "edit" adicione qualquer conteudo que queira.

![Imagem 11](https://files.cdn.thinkific.com/file_uploads/401289/images/1de/ddc/e13/1716930410833.png)

Agora vamos salvar o arquivo na branch pois iremos utilizar na próxima atividade. Clique em **Commit Changes**

![Imagem 12](https://files.cdn.thinkific.com/file_uploads/401289/images/581/eb0/4c7/1716930436777.png)

Após confirmar a operação teremos o novo arquivo salvo na branch.

## Criar e Gerenciar Issues

Objetivo: Aprender a criar e gerenciar issues no repositório.

Vá para a aba "Issues" no repositório `learn-github` e clique em "New issue".

![Imagem 14](https://files.cdn.thinkific.com/file_uploads/401289/images/b90/cb1/5f1/1716930559631.png)

Adicione o título "Adicionar README" e descreva a necessidade de um arquivo README. Clique em "Submit new issue".

![Imagem 15](https://files.cdn.thinkific.com/file_uploads/401289/images/232/7f1/a1c/1716930584784.png)

Explore a diferença entre Issue, Discussão e Pull Request clicando nas respectivas abas e criando um exemplo de cada um.

![Imagem 16](https://files.cdn.thinkific.com/file_uploads/401289/images/984/0ed/65e/1716930615819.png)

## Criar Discussions

Objetivo: Compreender as discussões e como elas se diferenciam das issues.

Acesse a aba Settings (Configurações) do seu repositório

Procure a opção "Discussions" rolando a tela para baixo e marque para ativar a função.

![Imagem 18](https://files.cdn.thinkific.com/file_uploads/401289/images/75b/2a0/c22/1716930693217.png)

Clique na aba "Discussions" e depois em "New discussion".

![Imagem 19](https://files.cdn.thinkific.com/file_uploads/401289/images/fc4/d3c/387/1716930707370.png)

Selecione categorias como "Announcements", "Ideas", "Polls", "Q&A" e "Show and tell" para iniciar discussões variadas.

![Imagem 20](https://files.cdn.thinkific.com/file_uploads/401289/images/002/ad9/c49/1716930722984.png)

Clique para criar uma pergunta ou ideia

![Imagem 21](https://files.cdn.thinkific.com/file_uploads/401289/images/dea/b5d/f9a/1716930759056.png)

Faça uma pergunta em sua discussão **ou** peça para um colega e responda. Marque sua resposta como a oficial usando "Mark as answer".

![Imagem 22](https://files.cdn.thinkific.com/file_uploads/401289/images/be1/837/eef/1716930851733.png)

## Trabalhar com Pull Requests

Objetivo: Aprender a criar e gerenciar pull requests.

Vá para a aba "Pull requests" e clique em "New pull request".

![Imagem 24](https://files.cdn.thinkific.com/file_uploads/401289/images/1e8/e90/537/1716931029831.png)

Selecione o branch base (onde as mudanças serão integradas) e o branch compare (de onde as mudanças vêm. Iremos usar a branch criada anteriormente, no inicio do exercicio). Clique em "Create pull request".

![Imagem 25](https://files.cdn.thinkific.com/file_uploads/401289/images/c1c/48b/136/1716931068249.png)

Adicione "Closes #1" na descrição para linkar a issue correspondente.

![Imagem 26](https://files.cdn.thinkific.com/file_uploads/401289/images/9ef/d6f/449/1716931095743.png)

Explore as abas "Conversation", "Commits", "Checks" e "Files changed" dentro do PR.

![Imagem 27](https://files.cdn.thinkific.com/file_uploads/401289/images/0a8/2bd/178/1716931113466.png)

Em seguida conclua o Pull Request clicando em "merge pull request"

## Criar e Gerenciar Wikis

Objetivo: Utilizar Wikis para documentar repositórios.

Vá para a aba "Wiki" no repositório e clique em "Create the first page".

![Imagem 29](https://files.cdn.thinkific.com/file_uploads/401289/images/c75/183/3f4/1716931158867.png)

Crie, edite e delete páginas de Wiki usando Markdown.

![Imagem 30](https://files.cdn.thinkific.com/file_uploads/401289/images/f81/ae1/3e3/1716931172260.png)

Entenda que a visibilidade das Wikis segue a configuração de privacidade do repositório.

![Imagem 31](https://files.cdn.thinkific.com/file_uploads/401289/images/629/ef7/6a7/1716931189904.png)
